---
description: デザインの修正からブラウザでの自動確認までを一括で行うワークフロー
---

// turbo-all

1. デザインに関連するファイル（`style.css`, `generate.py`, `src/sections/*.html` など）をユーザーの指示通りに修正します。
2. もし修正が `index.html` に直接反映されない形式（Pythonスクリプトによる生成が必要な場合など）であれば、生成スクリプトを実行します。
   ```powershell
   python generate.py
   ```
3. ブラウザサブエージェントを起動して、修正箇所を自動で見つけ出し、スクリーンショットを撮影して報告します。
   - 対象URL: http://localhost:63509
   - アクション: 修正したセクションまでスクロールし、表示を確認。
   - アウトプット: `design_update_check` という名前でスクリーンショットを保存。
4. 撮影したスクリーンショットを確認し、意図通りであれば完了を報告します。修正が必要な場合は 1 に戻ります。
